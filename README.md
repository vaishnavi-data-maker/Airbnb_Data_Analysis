# Airbnb_Data_Analysis
# Airbnb NYC Data Analysis Project

## 📌 Project Overview
This project performs **Exploratory Data Analysis (EDA)** on the Airbnb NYC dataset using Python.  
The goal is to analyze pricing, availability, room types, neighborhoods, reviews, and host activity to discover useful business insights.

The project includes:
- Data Cleaning
- Data Visualization
- Statistical Analysis
- Geographical Mapping using Folium
- Business Insights & Conclusions

---

# 📂 Dataset Information

Dataset contains Airbnb listings in New York City with features like:
- Listing Name
- Host Information
- Neighborhood
- Latitude & Longitude
- Room Type
- Price
- Minimum Nights
- Reviews
- Availability
